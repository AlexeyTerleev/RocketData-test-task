## RocketData (test task)

### Install

Clone repository:

```bash
git clone https://github.com/AlexeyTerleev/RocketData-test-task.git
```

Install dependencies:
```bash
cd RocketData-test-task
pip install -r requerements
```

### Run

```bash
python3 main.py
```
